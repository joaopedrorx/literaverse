# Literaverse

Uma plataforma web para leitores e escritores se conectarem, descobrirem e publicarem histórias.

## Funcionalidades

*   **Autenticação de usuário:** Usuários podem se registrar, fazer login e logout.
*   **Navegar na biblioteca:** Explore uma coleção de livros.
*   **Escrever e publicar:** Um editor WYSIWYG para escrever e publicar histórias.
*   **Rascunhos:** Salve rascunhos localmente no navegador.
*   **Alternador de tema:** Alterne entre temas escuro e claro.
*   **Design responsivo:** A aplicação é otimizada para diferentes tamanhos de tela.

## Tecnologias Utilizadas

*   **HTML:** A estrutura das páginas web.
*   **CSS:** Estilização da aplicação, com a ajuda do framework Bootstrap.
*   **JavaScript:** Implementação da funcionalidade da aplicação, como autenticação de usuário, alternador de tema e o editor.

## Como Executar

1.  Clone este repositório.
2.  Abra o arquivo `index.html` no seu navegador web.

**Nota:** Esta é uma aplicação apenas front-end. Todos os dados são armazenados localmente no `localStorage` do seu navegador.
